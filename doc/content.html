Fill Layer

This style is the main for filling the element with color. On the layers’ panel placed above the shadow and can be used to darken or lighten an image.

Action

Here example code using like in the above

<div class="ss__bg__secondary"></div>
<div class="ss__bg__info"></div>
<div class="ss__bg__success"></div>
<div class="ss__bg__warning"></div>
